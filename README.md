# multiPageApp

```bash
├── README.md
├── deploy // 产出脚本
├── gulpfile.js 
├── package.json
├── src
│   ├── assets
│   ├── components
│   ├── layout
│   │   └── head
│   │       ├── 111.jpg
│   │       ├── index.html
│   │       └── index.less
│   ├── pages
│   │   └── home
│   │       ├── index.html // 页面模板必须为 index.html
│   │       ├── index.js // 入口 js 必须为 index.js
│   │       └── index.less
│   └── utils
├── tsconfig.json
└── webpack.config.js
```
